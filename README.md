# Worker Contracts 🚀

Este é um sistema desenvolvido em **Java** para console que simula o gerenciamento de contratos de um trabalhador, aplicando conceitos fundamentais de Orientação a Objetos (OO) como composição e manipulação de datas.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **Controle de Versão:** Git / GitHub

## 🎯 Funcionalidades
* Cadastro de trabalhador com nome, nível de experiência e salário base.
* Associação do trabalhador a um departamento específico.
* Registro de múltiplos contratos por hora (data, valor por hora e duração).
* Cálculo dinâmico do rendimento total (salário + contratos) para um mês e ano específicos informados pelo usuário.

## 💡 Conceitos Aplicados
* Composição de objetos (relação Tem-Um e Tem-Muitos)
* Enumerações (`Enum`) para categorização de níveis
* Manipulação, formatação e filtragem de datas (`Date`, `Calendar` e `SimpleDateFormat`)
* Uso de listas dinâmicas (`List` e `ArrayList`) para gerenciar os contratos
